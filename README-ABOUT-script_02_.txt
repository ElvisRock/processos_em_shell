# Script de Verificação e Criação de Arquivo

Este script foi desenvolvido para solicitar ao usuário o nome de um arquivo, 
verificar se ele já existe, e realizar a criação do arquivo caso não exista. 
Se o arquivo for criado com sucesso, o script verifica se a operação foi bem-sucedida e 
exibe o arquivo no diretório.

## Funcionalidades

1. **Solicitar ao usuário o nome de um arquivo**: O script pede que o usuário insira o nome de um arquivo a ser verificado ou criado.
2. **Verificar se o arquivo já existe**: O script checa se o arquivo fornecido pelo usuário já está presente no diretório.
   - Se o arquivo já existir, o script informa ao usuário.
   - Se o arquivo não existir, o script cria o arquivo e informa ao usuário sobre a criação.
3. **Verificação de sucesso da criação**: Após criar o arquivo, o script confirma se a operação foi realizada com sucesso e, em seguida, exibe o arquivo no diretório.
