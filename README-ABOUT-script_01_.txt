# Script de Manipulação de Arquivos e Diretórios

Este script realiza uma série de operações em arquivos e diretórios, incluindo a criação de arquivos, a modificação de extensões, a contagem de linhas e o backup de arquivos. Abaixo estão as etapas que o script executa.

## Funcionalidades

1. **Criar um diretório**: O script cria um novo diretório.
2. **Adicionar arquivos vazios**: Dentro do diretório, o script cria três arquivos vazios.
3. **Adicionar conteúdo ao terceiro arquivo**: O script insere cinco linhas com qualquer frase no terceiro arquivo.
4. **Verificar a quantidade de arquivos no diretório**: O script conta e exibe a quantidade de arquivos no diretório.
5. **Alterar a extensão do segundo arquivo**: O script altera a extensão do segundo arquivo para outro formato de sua escolha.
6. **Renomear o terceiro arquivo**: O script renomeia o terceiro arquivo com um nome de sua escolha.
7. **Verificar a quantidade de linhas no terceiro arquivo**: O script usa o comando `wc -l` para contar e exibir o número de linhas do terceiro arquivo.
8. **Obter a primeira e a última linha do terceiro arquivo**: O script captura a primeira e a última linha do terceiro arquivo.
9. **Imprimir o conteúdo do terceiro arquivo com número de linhas**: O conteúdo do terceiro arquivo é exibido com o número de cada linha (número da linha).
10. **Escrever uma frase no primeiro arquivo**: O script escreve uma frase de sua escolha no primeiro arquivo.
11. **Concatenar o conteúdo do primeiro arquivo no terceiro arquivo**: O conteúdo do primeiro arquivo é concatenado ao final do terceiro arquivo.
12. **Mover o terceiro arquivo para a pasta de backup**: O script cria uma pasta chamada "backup" e move o terceiro arquivo para essa pasta.
