Objetivo: 
Criar um script Shell que verifique a validade de um número digitado pelo usuário. 
O script deve gerar mensagens de erro apropriadas e, 
dependendo do valor fornecido, realizar diferentes ações, incluindo a gravação dos dados em um arquivo.


Requisitos:
O script deve pedir ao usuário para digitar um número.

O script deve verificar:
1 - Se o valor fornecido não for 8, deverá exibir uma mensagem de erro.
2 - Se o número for 100, deverá exibir outra mensagem de erro e encerrar a execução.

3 - Se o número for 589, deverá exibir uma mensagem de sucesso, dizendo que o número é maior que 500, 
solicitar ao usuário o "seu nome" e exibir uma mensagem de parabéns. 
Depois, deverá gravar os dados (número e nome) em um arquivo delimitado por pipe (|).

Caso a gravação seja bem-sucedida, o script deve exibir uma mensagem de sucesso. Caso contrário, 
deve exibir uma mensagem de erro.
