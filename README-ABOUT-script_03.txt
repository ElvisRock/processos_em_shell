Exercício: Manipulação Simples de Arquivos e Diretórios
 
Crie um script que:
 
       obs: Trabalhar em um diretório especifico
 
1 - Solicite ao usuário um nome de um arquivo ou pasta.
2 - Verifique se o nome é de um arquivo ou um diretório.
3 - Se for um arquivo  e não está vazio :
   Exiba uma mensagem dizendo que é um arquivo.
4 - Se for um diretório:
 
   Exiba uma mensagem dizendo que é um diretório. 
   Liste o conteúdo dele. 
5 - Se não existir, ofereça a opção de criar:
 
   Um arquivo vazio ou
   Um diretório.
